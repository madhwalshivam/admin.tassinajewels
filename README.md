# Tassina Jewellers - Admin & Storefront

This repository contains the source code for the **Tassina Jewellers** storefront custom integrations and administration portal.

## Project Structure

- `theme.liquid`: The core customized Shopify/Shoplazza theme file containing standardized custom slider navigation and interactive components.
- `/admin`: Next.js administration portal, configured to connect to Supabase for customer reviews, dynamic deals, and product categorizations.

## Technologies Used

- **Storefront**: Liquid, Vanilla JS, Swiper.js, CSS
- **Admin Portal**: Next.js, React, Tailwind CSS, Supabase (PostgreSQL)
